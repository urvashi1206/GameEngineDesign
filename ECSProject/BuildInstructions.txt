Run the project on x64 Native Tools Command Prompt for VS 2022

In VSCODE press CTRL+SHIFT+P
In the include path add "C:/Users/Owner/Documents/GameEngineDesign/vcpkg/installed/x64-windows/include"

To build : 
rmdir /S /Q build
mkdir build
cd build
cmake .. -G "Ninja" -DCMAKE_BUILD_TYPE=Debug ^
  -DCMAKE_TOOLCHAIN_FILE="C:/Users/Owner/Documents/GameEngineDesign/vcpkg/scripts/buildsystems/vcpkg.cmake"
ninja

To run:
.\bin\ECSProject.exe
